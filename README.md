# JobSearch
Dự án "Manual Testing for Job Search Website" kiểm tra chất lượng và tính năng của trang web tìm kiếm việc làm. Công việc kiểm thử thủ công sẽ tập trung vào kiểm tra tính năng tìm kiếm, đăng ký, ứng tuyển và các chức năng khác, nhằm phát hiện lỗi và đảm bảo trải nghiệm người dùng mượt mà, chính xác và ổn định.
